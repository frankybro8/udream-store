# Udream Store Website

A responsive static storefront for Udream, built with HTML, CSS, and JavaScript. Now in Arabic with cart functionality.

## Files
- `index.html` — main website page (Arabic)
- `styles.css` — page styling with RTL support
- `script.js` — customer chat widget and cart logic

## Features
- Arabic language interface
- Add to cart buttons on products
- Cart widget with item management
- Customer chat support

## Preview
Open `index.html` in your browser to view the site.

## Customize
- Replace the hero image and product images in `index.html` with your own logo and background.
- Adjust colors and text in `styles.css`.
- Connect the chat widget to a real backend or chatbot service later in `script.js`.
- Cart data is stored in localStorage for persistence.
